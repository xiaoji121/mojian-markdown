Workflow

从打开文档，到形成自己的理解。

01

直接编辑，实时生效

Markdown 原文和预览同步更新，预览区也可以直接编辑，接近所见即所得。

02

双击双向定位

双击 Markdown 内容可定位预览位置；双击预览内容，也能回到对应原文。

03

划线批注

选中文字后添加马克笔、波浪线、直线或想法，把评注集中收进批注面板。

04

本地缓存，随时继续

当前文章、文件名、字号、主题和批注会保存在浏览器本地，下次打开页面可以继续阅读。

For Study

让 Markdown 变成可以评注的学习材料。

墨笺适合阅读方案、接口文档、论文笔记和课程材料。你可以在原文上留下疑问、理解和反例，把一次阅读变成后续可回看的学习材料。

原文用户无法切换组织管控
批注这里需要确认权限边界和灰度范围。
整理复制全文和批注后，可继续放进自己的知识库或 AI 工具里处理。
Open In Browser

打开网页就能用，内容留在你的浏览器里。

官网版是纯静态应用，不需要 Node、Agent Bridge 或 Claude。文章和批注会写入浏览器 localStorage，重要内容建议用“保存”或“复制全文+批注”导出备份。
