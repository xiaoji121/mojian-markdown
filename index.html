在线体验编辑与批注 GitHub 开源仓库 ↗

当前站点是线上体验版：包含编辑、预览、批注和本地缓存；左侧最近阅读列表与 AI 问答依赖本地 Agent Bridge，需本地运行开源完整版。

MIT 开源 无需安装 浏览器本地缓存 可导入导出
Workflow

从打开文档，到形成自己的理解。

01

直接编辑，实时生效

Markdown 原文和预览同步更新，预览区也可以直接编辑，接近所见即所得。

02

双击双向定位

双击 Markdown 内容可定位预览位置；双击预览内容，也能回到对应原文。

03

划线批注

选中文字后添加马克笔、波浪线、直线或想法，把评注集中收进批注面板。

04

本地缓存，随时继续

当前文章、文件名、字号、主题和批注会保存在浏览器本地，下次打开页面可以继续阅读。

05 完整版

选中即问 AI

选中一段原文向 AI 提问，回答会结合整篇文档；每次问答自动归档为该文档的子节点。

06 完整版

最近阅读列表

本地 Agent Bridge 维护阅读工作区：读过的文档和问答记录都在左侧列表里，随时重新打开。

For Study

让 Markdown 变成可以评注的学习材料。

墨笺适合阅读方案、接口文档、论文笔记和课程材料。你可以在原文上留下疑问、理解和反例，把一次阅读变成后续可回看的学习材料。

原文用户无法切换组织管控
批注这里需要确认权限边界和灰度范围。
整理复制全文和批注后，可继续放进自己的知识库或 AI 工具里处理。
